# 特质波动率与下月收益预测

状态：开题并预注册；尚未计算本题结果。

## 问题

在 25 个固定、成熟、当前市场质量较高的 Binance USD-M perpetual 中，过去 90 个完整 UTC 日收益对“排除自身的等权市场收益”回归后的残差波动率（IVOL90），是否负向预测下一个 UTC 月 open-to-open 收益？这一关系在控制总波动率、MAX28、MOM90、beta 和成交额后是否仍在，并且是否为个人小资金的月频单腿 one-shot 计划留出现实成本余量？

本题是 `PREDICTIVE`，不直接产生策略或 handoff。预测门全部通过后，才允许另建 `STRATEGY_CANDIDATE` 问题处理 funding、fee、spread/slippage、保证金、持仓路径和框架无关交付。

## 为什么选它

开题前将少量方向按项目决策价值、未解差异、可证伪性、数据与执行成本排序：

| 方向 | 判断 | 本轮决定 |
|---|---|---|
| 特质波动率→下月收益 | 2020、2021、2026 研究对正负方向和 MAX 解释存在直接冲突；非微型币的最新证据给出负向预测；可用已校验日线快速证伪 | **选中** |
| UTC 日内/日历季节性 | 多重比较空间大，近期证据不稳；人工激活窗口增加执行摩擦 | 未选 |
| PAXG/XAU 趋势转换 | PAXG 单标的简单趋势已失败；XAU/XAG perpetual 历史短且需原市场日历/basis 语义 | 未选 |
| BTC/ALT 配对或相对价值 | 当前机会前沿审计已在成本后否定简单 stat-arb，且双腿对半自动计划更复杂 | 未选 |

本题不是把已失败的“总波动率高端做空”换参数重开。它事前要求 IVOL 在控制总波动和 MAX 后仍有负向增量，并直接与原始高波动分组比较。
