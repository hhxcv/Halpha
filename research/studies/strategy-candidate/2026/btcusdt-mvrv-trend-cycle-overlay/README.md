# BTC MVRV 底部门控与趋势退出候选

## 问题与候选

固定 MVRV 复核只支持开发期底部附近入场，不支持 5/6/7 作为独立顶部卖出。下一项最小复杂度候选因此只组合两类已有证据：MVRV 低估交叉负责启动新周期，长期 SMA200 只在 MVRV 高估后负责退出和重新进入；1.25x 上限用于补偿趋势退出无法命中事后最高点的结构性差距。

候选 `BTC_MVRV_BOTTOM_TREND_EXIT_ENSEMBLE_1P25` 分为三个等权袖套，退出警戒阈值固定为 5/6/7。每个袖套在 MVRV expanding Z 从上向下穿过 -0.2 后进入 1.25/3；在各自 Z 从下向上穿过顶部阈值后进入“趋势管理”，收盘价高于 SMA200 时仍持有 1.25/3，低于或等于 SMA200 时为现金。新的 -0.2 底部交叉会重置三个袖套的顶部警戒。信号只用已完成日数据，下一日行动；不做空，最高总敞口 1.25x。

三个诊断不参与选优：相同组合最高 1.0x、MVRV 底部后持续 1.25x、单独 SMA200 最高 1.25x。主基准是同起点、同成本 1x buy-and-hold，且逐牛段直接比较原始总收益。

## 选择偏差声明

本候选在看过 2014～2021 BTC 价格、MVRV 5/6/7 复核和先前慢趋势失败后形成，不是对市场历史的首次盲测。5/6/7 采用等权而不选择开发期冠军；仍把既有 Donchian、慢趋势和 MVRV 规则计入 30/60 次相关试验敏感性。2022～2026 只算规则级顺序确认：此前已看过该段价格和当前 MVRV，不能称为完全未见数据。

## 数据、时期与成本

- Coin Metrics `PriceUSD`、`CapMVRVCur`、`CapMrktCurUSD`，2010-07-18～2026-07-01；先固定原始哈希和内容身份。
- development：2014-01-01～2022-01-01；confirmation：2022-01-01～2026-07-01，只有 development 全门通过才启封。
- 周期评分窗固定为 2014-12～2015-03 / 2017-11～2018-02、2018-11～2019-03 / 2021-10～2022-01、2022-10～2023-02 / 2025-09～2025-12；端点不进入信号。
- Favorable/Base/Stress 每单位换手成本 10/25/50 bp；对候选全部名义敞口施加 0%/5%/10% 年化融资代理。该融资口径比只对 0.25x 借款计息更保守；buy-and-hold 不承担融资。
- 权重相对偏差超过 20% 或目标跨零才再平衡；终止时计平仓成本。

## 运行前反证门

Development 必须同时满足：两个完整牛段在 Base 和 Stress 下都超过 1x buy-and-hold；全开发期 Base/Stress 都超过持有；最大回撤更浅；完整 2017 高点到 2018 低点熊段少亏；两个低点附近 ±90 日内有 MVRV 重置且高点当日仍至少 1x；两个高点后 180 日内降至不高于 0.25x；不被任一简单诊断在收益、Sharpe、回撤三项严格支配；30 次试验 DSR 至少 0.80；180 日成块自助中超过持有的概率至少 0.80。

任一失败即停止，不打开 confirmation，也不从诊断中替换候选。若全门通过，confirmation 仍须逐牛段 Base/Stress 超过持有、全期超过持有、回撤更浅、2025 高点后观察段少亏，并在独立 Binance 实际 funding 桥接中通过；否则不得进入 Demo。

产品作用为 `NONE`，本研究不创建计划、不连接账户、不执行交易。
