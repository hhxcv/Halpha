# 结果：开发期有正收益，但证据不足以进入评价或 Demo

结论为 `INSUFFICIENT_EVIDENCE`。这不是“Donchian 无效”，也不是“找到可上线策略”：6 个事前固定方案在 2021–2023 的 favorable/base/stress 下都为正，但没有一个同时满足预注册的相对基准和选择偏差门。`evaluation_authorized=false`，所以 2024–2025 与 2026H1 未被本问题读取。

描述上最接近的是 `LONG_BALANCED_4`（20/30/60/90 日 long-only）。它的三年 base 年度结果为 +2.11%、-5.10%、+11.84%，Calmar 0.322，四个单周期分量均为正。但它的 Sharpe 0.473 只略高于相同 10% 波动目标、0.5x 上限的持续多头基准 0.469；六候选修正后的 DSR 0.750 低于事前固定的 0.80。现在挑出其中表现最好的 60 日单周期或把 DSR 门降到 0.75，都会是看过结果后的选择，因此没有这样做。

持续多头基准同期 base 为 +13.92%、最大回撤 -18.63%、Calmar 0.238。Donchian 组合的实际价值更像以较低收益换取更浅回撤，而不是被验证的超额收益；这还不足以增加第二个正式策略。

## 差距不主要在交易成本

`LONG_BALANCED_4` 的 favorable/base/stress 分别为 +8.86%/+8.38%/+8.08%。把 20% 容忍带改为每日调整，base 只变为 +8.46%；说明结论不依赖一个脆弱的换仓阈值。零交易成本但保留真实 funding 时为 +9.21%，保留 base 成本但移除 funding 时为 +13.44%。三年 funding 拖累约为初始资本的 4.72 个百分点，是比手续费与滑点更大的场所差异。

论文式 25% 波动目标、2.0x 上限的完整九周期诊断得到 +15.82%，但 Sharpe 只有 0.422、最大回撤扩大到 -23.51%、换手显著增加。它只是放大风险和名义收益，没有修复风险调整优势，因此不作为产品候选。

## 为什么公开研究可以得到更强结果

公开结果与 Halpha 当前检验并不矛盾：

- 论文的 BTC 结果覆盖 2015–2025，包含 Binance 永续不存在的早期强趋势；Halpha 可执行场所从 2020 开始，开发门只看 2021–2023。
- 论文最强的扣费后结论来自动态 top-20 多币组合。趋势跟随的长期证据依赖跨市场和跨周期分散；单一 BTC 没有继承这一分散收益。
- 论文 BTC 单资产表未扣交易成本，且使用最高 2.0x 的 25% 波动目标。Halpha 以实际 funding、市场单成本和 0.5x 产品边界重新计算。
- 公开方案把 Donchian 用作完整的持仓状态与单向跟踪退出，允许少数趋势延伸；当前正式策略只把 Donchian 用于 15 分钟一次入场，随后固定止盈并很快退出。二者不是同一策略。
- 即使采用公开方案，本地开发期也只是“有收益”，并未显示足以区别于波动匹配 BTC beta 的可靠优势。

## 复核与边界

开发 CSV 和每日回报在 Git 外独立目录重演后 SHA-256 逐字节一致；JSON 去除生成时间后语义一致。24 个开发期 funding 事件因 Binance 官方 mark-price 时间轴缺口使用日开盘价，改用日收盘价后总收益只变化 `0.0000002665`，不影响门或结论。

因此不创建 Demo 计划、不实现产品策略，也不打开评价/确认期。这样做正是为了兑现“充分验证之前不草率下结论”：Demo 应验证一个已有历史支持的候选的在线语义，不应被用来替历史筛选寻找正结果。
