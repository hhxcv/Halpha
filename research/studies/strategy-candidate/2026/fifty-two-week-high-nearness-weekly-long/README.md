# 52 周高点接近度周频单腿 LONG

本目录研究一个适配 Halpha 半自动策略计划的框架无关候选：固定 25 个当前长期、较高活动 Binance USD-M 永续目标；每周只使用截至周六的完整公开日线，按 `Nearness52 = 周六收盘 / 最近 52 个周六收盘的最高值` 排名。用户固定目标只有进入顶部十分位时，才在周一 open 提出一次 `0.25x LONG / 7d` 计划；周日构成一个完整的信号—行动间隔。

问题来自 Jia 等 2026 年同行评审的加密货币 52 周高点锚定研究。原文报告 Nearness52 与下一周横截面收益正相关，但其主要证据是宽市场现货多空组合，不包含 Binance 永续 funding、个人零售成本、固定存续目标、单腿绝对收益、one-shot 冷却或人工激活。本文只检验这项运营转换，并把普通 52 周动量和同周市场长仓设为硬基准。

规则、顺序阶段、否定条件和支持上限见 `preregistration.md`；来源、时间与适用差异见 `sources.md`；全部选择、尝试、失败和修正按发生顺序写入 `attempts.md`。大型公开行情继续复用 Git 外、带 SHA-256 身份的 Binance 缓存，不复制进本目录。

本研究不修改产品代码、正式策略、L4、资金或真实账户状态，不启动产品运行时，不使用凭据或产品业务数据，也不调用任何交易所变更端点。
