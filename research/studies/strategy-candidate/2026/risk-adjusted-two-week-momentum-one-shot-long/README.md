# RMOM2 周频顶部五分位单腿 LONG

本目录研究一个框架无关候选：在固定的 25 个当前长期、较高活动 Binance USD-M 永续中，以截至周日收盘的过去 14 个完整日收益均值除以样本波动率（`RMOM2`）排名；用户固定目标只有位于顶部五分位时，才提出一次 `0.5x LONG / 7d` 计划。

它不是正式策略，也不修改交易核心。研究必须证明它在现实费用、funding、spread/slippage、全计划资本门槛后，优于原始两周动量、单纯低波和定时做多；否则风险调整只是同一动量或低波暴露的重新表达。

主要文件：

- `preregistration.md`：候选筛选、唯一主规则、数据暴露、顺序门与否定条件。
- `sources.md`：原始因子研究、近期反证、官方数据和框架资料。
- `study.py`：检查点、数据身份、RMOM2、计划、成本、门控和验证。
- `attempts.md`：所有重要尝试、失败和修订，按发生顺序追加。
- `source_reuse_manifest.json`：复用上一问题公开 Binance 数据和外部论文 PDF 的身份。
- `checkpoint.json`：任何 RMOM2 收益输出前冻结的问题、代码和门槛。

大型公开数据继续位于既有 Git 外缓存；本题不复制数据，也不接触产品数据、数据库、凭据或运行时。
