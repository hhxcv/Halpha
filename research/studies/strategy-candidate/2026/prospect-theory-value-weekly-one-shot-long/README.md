# 前景理论价值低分周频单腿 LONG

本题检验一个适合个人、小资金、半自动策略计划的行为定价候选：每周只使用截至周六的公开日线，把固定流动 Binance USD-M 永续目标最近 52 个周收益相对同期等权市场的分布，按 Tversky–Kahneman 累积前景理论参数压缩为 `PTV52`。用户固定目标只有进入横截面最低十分位时，才在周一 open 提出一次 `0.25x LONG / 7d` 计划；完整周日不参与信号。

问题来自 Chen、Lepori、Tai 与 Sung 2022 年同行评审论文。原文用 1,573 个现货币的宽市场、点时对象和市值加权参考，报告更高 PTV 对下一周相对收益有负向预测；本题只检验它能否转成固定幸存永续名单、单腿绝对收益、现实成本/funding、one-shot 冷却与人工激活计划。普通输家、低偏度、高波动和市场多头均为硬基准。

大型公共行情只保存在 Git 外的既有 Binance 缓存，本目录保留来源身份、冻结规则、代码、逐笔 CSV、摘要与失败。研究不修改产品代码、正式策略、L4、资金或账户，不使用凭据或产品业务数据，不启动产品运行时。
