# 结论：不支持只靠入场选择性改出正净期望

结论为 `DOES_NOT_SUPPORT`。在固定当前 1.5 ATR 止损、50% @ 1.5R、余量 @ 3R 和最长 4×15m 的前提下，开发期完整 72 组 `lookback × confirmation × extension × direction` 没有任何一组取得正的 favorable、base 或 stress 平均净收益。开发门通过数为 0，按预注册顺序停止；2024–2025 evaluation 和 2026H1 confirmation 均未读取。

## 关键结果

| 配置 | 情景 | 笔数 | 平均净收益/笔 |
|---|---:|---:|---:|
| base 最佳 LONG 64/2/0.5 | base | 943 | -0.2904% |
| favorable 最佳 LONG 64/3/0.25 | favorable | 672 | -0.1244% |
| 正式默认 LONG 20/2/0.5 | base | 2,049 | -0.3247% |
| 正式默认 SHORT 20/2/0.5 | base | 1,981 | -0.3686% |
| 历史激进 LONG 4/1/0.5 | base | 8,175 | -0.3051% |
| 历史激进 SHORT 4/1/0.5 | base | 8,063 | -0.2995% |

LONG 64/2/0.5 相对正式 LONG 默认改善约 3.42 bps/笔，但 2021、2022、2023 的 base 均值分别仍为 -0.2985%、-0.3131%、-0.2414%。改善不足以接近开发门，更不授权把它当成新默认。

## 可采取的收敛

1. 不继续搜索通道回看、确认根数和追价边界的近邻值；已锁定族全部失败。
2. 不同时放开止损、两档止盈和持仓期做大网格；这会扩大选择偏差，且没有先行毛边际支持。
3. 下一研究若继续，应提出不同的可证伪问题。最直接的是“60 分钟退出是否截断了趋势机制”，但它必须单独锁定少量更长持仓期、funding 与尾部风险门，不能在本问题已暴露数据上追逐最优。
4. 当前自然信号 Demo 激活仍可用于执行链路和真实费用证据，但单笔盈亏不能推翻本结论。

限制：1m OHLC、持续重武装代理、人工激活日程缺失和已暴露的底层市场数据，使本结果只能否定当前小型参数族，不能外推为所有 Donchian、所有持仓期或所有市场中的趋势无效。
