# BTC 大周期慢趋势分层敞口研究

## 固定问题

用户希望下一轮 BTC 牛市从低点到高点的策略净收益超过持续 1x buy-and-hold，同时在牛熊大周期中把握少量中期波段。由于未来低点和高点只能事后知道，本题不预测一个精确价位，而检验：一个仅使用当时已完成日线、由成熟慢趋势信号组成、最大 1.5x 的固定分层敞口，能否在三个历史牛段的严格“事后低点到事后高点”基准下逐段超过 1x 持有，并在熊段和全周期保留优势。

- 主要类型：`STRATEGY_CANDIDATE`。
- 候选：`BTC_MACRO_SCORE_3_MAX_1P5`。
- 最强允许主张：若全部顺序门通过，只支持该低复杂度机制继续作为 BTC 候选研究；不证明当前已经见底、未来高点、未来盈利或产品可用性。
- 产品作用：`NONE`；本题不进入 Demo，不创建计划，不修改产品。

## 为什么这是新问题

已有 Halpha BTC 研究主要检验低 gross Donchian、90 日 long/cash、8h 执行和短周期 one-shot 入场。它们以降低回撤或取得绝对正收益为目标，最大权重通常不超过 0.5，因此在牛市低点到高点不可能系统性满足“超过 1x 持有”的新目标。论文式 25% 波动目标/2.0x Donchian 只作为旧研究诊断，且在 2021–2023 没有修复风险调整优势。

本题不继续扫 Donchian 周期。它使用三个外部文献锚定的慢信号，并用等额投票逐步改变敞口：

1. 收盘价高于 200 日简单均线；
2. 50 日简单均线高于 200 日简单均线；
3. 84 日（12 周）收益为正。

每个成立信号贡献 0.5x，目标敞口只能为 0、0.5、1.0 或 1.5。日线收盘后形成目标，下一日行动；目标与实际权重相对偏差不超过 20% 时不因漂移再平衡。最大 1.5x 是为严格牛段超越 1x 基准所必需的有限战术上限，不搜索 1.25/1.75/2.0；无杠杆缩放和单一 SMA200 只作反解释，不能晋升候选。

## 数据与时间顺序

- 长历史机制：Coin Metrics Community `BTC PriceUSD`，UTC 日频，2013-01-01～2026-06-30；2013 只作 200 日 warmup。
- development：2014-01-01～2021-12-31，只启封 2015–2017 与 2018–2021 两个历史牛段。
- confirmation：2022-01-01～2026-06-30，仅在开发门通过后启封，包含 2022–2025 牛段及其后的已观察熊段。
- 场所桥接：Binance USD-M BTCUSDT perpetual 2020-01-01～2026-06-30，复用已校验的官方日线、8h mark price 与实际 funding；只在 confirmation 获准后运行。
- 当前状态：2026-07-21 的官方 Binance 完整日线与 Coin Metrics MVRV 另作描述性快照，不进入历史规则选择。

全部 BTC 历史价格和许多相关趋势结果已在仓库其他研究中暴露。本题的精确三信号映射、严格逐牛段目标和顺序输出此前未运行；它是规则级顺序证据，不是全新市场数据留出。

## 事后周期标签与公平基准

周期端点只用于评分，绝不进入信号。每个端点由预先固定的日历窗口内最低/最高 Coin Metrics 日价格确定：

| 周期 | 低点搜索窗 | 高点搜索窗 |
|---|---|---|
| 2015–2017 | 2014-12-01～2015-03-01 | 2017-11-01～2018-02-01 |
| 2018–2021 | 2018-11-01～2019-03-01 | 2021-10-01～2022-01-01 |
| 2022–2025 | 2022-10-01～2023-02-01 | 2025-09-01～2025-12-01 |

低点形成后的下一日开始计量，到高点日收盘为止。候选只能使用低点当日已经可得的目标状态；buy-and-hold 则被有利地假定为在低点后的第一日买入 1x，因此这是对候选更严格而非更宽松的比较。

## 现实成本与反证

Coin Metrics 长历史只能验证价格机制，使用每单位换手 10/25/50 bp，候选另对全部名义敞口收取 0%/5%/10% 年化融资代理；现金收益为零。该代理比只对超过 1x 的部分收费更保守。Binance 桥接采用原研究已冻结的市场单成本和全部实际 funding。

开发期必须同时满足：

- 两个牛段在 Base 和 Stress 下均超过各自 1x buy-and-hold；
- development 全期 Base 和 Stress 均超过 buy-and-hold，最大回撤更浅；
- 2017 高点至 2018 低点的完整熊段超过 buy-and-hold；
- 两个低点后恢复至少 1.0x 的滞后均不超过 180 日，2017 高点后降到不高于 0.5x 不超过 120 日；
- 不被同为 1.5x 上限的简单 SMA200 规则严格支配；
- 计入 20 个相关历史尝试的 DSR 至少 0.80；
- 180 日配对循环块自助中，全期总收益超过 buy-and-hold 的概率至少 80%。

任一失败即停止并保持 confirmation 未启封。若开发通过，confirmation 还必须让 2022–2025 牛段 Base/Stress 均超过 buy-and-hold、2025 高点后的已观察熊段更优、全段回撤更浅，并通过相同 Binance 实际 funding 桥接。即使全部通过，三个历史周期仍不足以证明下一轮，结论最多为有边界的 `SUPPORTS_WITHIN_SCOPE`。

## 运行边界

研究使用 VectorBT 的收益统计与 DSR，并复用既有 funding-aware 线性永续模拟；Halpha 只增加本题三个信号、周期评分、块自助和结果导出，不自建回测框架。实际命令与哈希在 checkpoint 和 `attempts.md` 固化。
